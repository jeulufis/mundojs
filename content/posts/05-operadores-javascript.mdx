1. Operadores Aritméticos
2. Operadores de Asignación
3. Operadores de Cadena
4. Operadores de Comparación
5. Operadores Lógicos
7. Operadores Especiales

### Ejemplos de Operadores Aritmeticos

 let a = 10;

console.log("Incremento: " + a); // 11

// Decremento
b--;
console.log("Decremento: " + b); // 4

## Operadores de Asignacion

<CodeBlock language="javascript">
  {` x = y
`}
</CodeBlock>

### Ejemplos de Operadores de Comparacion

<CodeBlock language="javascript">
  {` let a = 10;
 let b = '10';
 let c = 20;

 console.log(a == b); // true, '10' se convierte a 10

 console.log(a != c); // true, 10 no es igual a 20

 console.log(a === b); // false, 10 y '10' son de diferente tipo

 console.log(a !== b); // true, 10 y '10' son de diferente tipo

 console.log(c > a); // true, 20 es mayor que 10

 console.log(a >= b); // true, 10 es igual a '10' después de la conversión de tipo

 console.log(a < c); // true, 10 es menor que 20

 console.log(a <= b); // true, 10 es igual a '10' después de la conversión de tipo
`}
</CodeBlock>

### AND

<CodeBlock language="javascript">
  {` console.log(true && true);    // true
 console.log(true && false);   // false
 console.log(5 > 3 && 8 < 10); // true, ambas condiciones son verdaderas
 console.log(5 > 3 && 8 > 10); // false, porque una de las condiciones es falsa
`}
</CodeBlock>

### OR

<CodeBlock language="javascript">
  {` console.log(true || false);   // true
 console.log(false || false);  // false
 console.log(5 > 3 || 8 > 10); // true, porque una de las condiciones es verdadera
 console.log(5 < 3 || 8 > 10); // false, porque ambas condiciones son falsas
`}
</CodeBlock>

### NOT

<CodeBlock language="javascript">
{` console.log(!true);   // false
 console.log(!false);  // true
 console.log(!(5 > 3)); // false, porque 5 > 3 es true, y !true es false
 console.log(!(5 < 3)); // true, porque 5 <CodeBlock 3 es false, y !false es true
`}
</CodeBlock>

## Operadores Especiales

### Operador Ternario

El operador ternario es el único operador JavaScript que toma tres operandos. Este operador se utiliza como una forma abreviada de una declaración if...else.

<CodeBlock language="javascript">
  {` let edad = 18;
 let status = (edad >= 18) ? 'adulto' : 'menor de edad';
 console.log(status); // adulto
`}
</CodeBlock>

### Operador de tipo de dato `typeof`

El operador `typeof` devuelve una cadena que indica el tipo de dato de una variable.

<CodeBlock language="javascript">
  {` let x = 10;
 console.log(typeof x); // number

 let y = 'Hola';
 console.log(typeof y); // string

 let z = true;
 console.log(typeof z); // boolean
`}
</CodeBlock>
