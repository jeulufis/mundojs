Vamos a entender la lógica detrás de cada de estas instrucciones. Para esto te recomiendo visitar la página
[jsconsole](https://jsconsole.com/). Este sitio nos facilitara el trabajo y simulara una consola en línea donde puedes ejecutar tu código JavaScript.

Te invito a ingresar a la página y probarlo por ti mismo. De esta manera, no solo podrás observar su funcionamiento, sino también familiarizarte con cómo interactúa y responde a diferentes entradas.

Para crear nuestro clásico primer **"Hola, mundo"** (se utiliza generalmente para mostrar en un ejemplo sencillo la sintaxis de un lenguaje de programación), es importante entender conceptos como `alert` y `console.log`.

## Alert

`alert`: Mostrara un diálogo de alerta con un **mensaje opcional**, y aguardará hasta que el usuario cierre la ventana de diálogo apretando el boton `OK`.

**Escribe el siguente codigo:**

```js
alert('Hola mundo! 👋');
```

<Image src="/post-2/hola-mundo.png" width="718" height="404" alt="Gráfico de uso de JavaScript" />

## Console.log

`console.log`: Muestra un mensaje en la consola del navegador o del intérprete de JavaScript. Es una herramienta útil para **depurar código, permitiendo rastrear y diagnosticar problemas** de manera más eficiente durante el desarrollo.

**Escribe el siguente codigo:**

```js
console.log('Hola mundo! 👋');
```

<Image src="/post-2/console-log.png" width="718" height="404" alt="Gráfico de uso de JavaScript" />

También puedes ejecutar todo esto directamente en la consola de tu navegador. Así es, la mayoría de los navegadores tienen su propia consola. Para abrirla, usa `Cmd + Opción + J en una Mac o Ctrl + Shift + J en Windows`. 

Si ves una pantalla similar a la siguiente, has abierto la consola correctamente, y podrás hacer el ejercicio de  escribir tus comandos `alert` y `console.log`.
