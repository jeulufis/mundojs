Al ocupar JavaScript es muy comun ocupar estos llamados operadores, estos operadores nos permitiran realizar operaciones matematicas con sus valores y comparar diferentes variables. En JavaScript tenemos estos tipos de operadores:

1. Operadores Aritméticos
2. Operadores de Asignación
3. Operadores de cadena
4. Operadores de comparación
5. Operadores lógicos
6. Operadores Bitwise
7. Operadores especiales

### Ejemplos de Operadores Aritmeticos

<CodeBlock language="javascript">
{` // Declaración de variables
let a = 10;
let b = 5;

// Suma
let suma = a + b;
console.log("Suma: " + suma); // Resultado: 15

// Resta
let resta = a - b;
console.log("Resta: " + resta); // 5

// Multiplicación
let multiplicacion = a * b;
console.log("Multiplicación: " + multiplicacion); // 50

// División
let division = a / b;
console.log("División: " + division); // 2

// Módulo (resto de la división)
let modulo = a % b;
console.log("Módulo: " + modulo); // 0

// Incremento
a++;
console.log("Incremento: " + a); // 11

// Decremento
b--;
console.log("Decremento: " + b); // 4

`}
</CodeBlock>
